# Handle Address Validate

Handle Address Validate

## Usage

``` r
handle_address_validate(db, config)
```

## Arguments

- db:

  A database connection pool created by
  [`pool::dbPool()`](http://rstudio.github.io/pool/reference/dbPool.md)

- config:

  The path to a `config.yml` file to be ingested by `{config}`

## Value

A handler function for route /address/validate
